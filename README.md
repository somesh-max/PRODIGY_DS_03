# Task 03 – Decision Tree Classifier

## Objective

Build a Decision Tree Classifier to predict whether a customer will purchase a product or service using the Bank Marketing dataset.

## Technologies Used

* Python
* Pandas
* Scikit-learn
* Matplotlib

## Steps Performed

1. Loaded the Bank Marketing dataset.
2. Preprocessed categorical features using one-hot encoding.
3. Split the dataset into training and testing sets.
4. Trained a Decision Tree Classifier.
5. Evaluated the model using accuracy and a classification report.
6. Visualized the trained decision tree.

## Result

The model achieved approximately **90.6% accuracy** on the test data and successfully predicted customer purchase behavior.
