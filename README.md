# 🌳 PRODIGY INFOTECH – Data Science Internship Task 03

## 📌 Project Title

**Customer Purchase Prediction using Decision Tree Classifier**

## 📖 Overview

This project builds a **Decision Tree Classifier** to predict whether a customer will purchase a product or service based on demographic and behavioral information. The analysis uses the **Bank Marketing Dataset** and applies machine learning techniques to classify customer responses accurately.

## 🎯 Objectives

* Build a Decision Tree Classification model for customer purchase prediction.
* Preprocess and encode categorical features for model training.
* Split the dataset into training and testing sets.
* Train and evaluate the Decision Tree model.
* Measure model performance using accuracy and classification metrics.
* Visualize the Decision Tree to understand the decision-making process.

## 🛠️ Technologies Used

* Python
* Pandas
* Scikit-learn
* Matplotlib
* Jupyter Notebook

## 📂 Dataset

The analysis uses the **Bank Marketing Dataset**, which contains customer demographic and campaign-related information such as age, job, marital status, education, balance, housing loan, contact type, campaign details, and whether the customer subscribed to a term deposit.

## 🔍 Key Insights

* The Decision Tree model effectively classifies customers based on demographic and campaign features.
* Customer attributes such as **duration, previous campaign outcome, balance, and contact type** significantly influence predictions.
* The trained model achieved an accuracy of approximately **90.6%** on the test dataset.
* The visualization helps interpret how different features contribute to the final prediction.
* Decision Trees provide an easy-to-understand and interpretable machine learning approach for classification tasks.

   ```bash
   pip install pandas matplotlib scikit-learn
   ```

3. Open the Python script or Jupyter Notebook.
4. Run all cells or execute the script to train the model and generate predictions.
5. View the Decision Tree visualization and evaluation metrics.

## ✅ Outcome

This project demonstrates how a **Decision Tree Classifier** can be used to predict customer purchase decisions using demographic and behavioral data. It showcases data preprocessing, feature engineering, model training, evaluation, and visualization, providing valuable insights into customer behavior and machine learning classification techniques.

## ✅ Result

The model achieved approximately 90.6% accuracy on the test data and successfully predicted customer purchase behavior.
